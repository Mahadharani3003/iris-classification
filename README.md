# Iris Flower Classification 

## Project Overview
This project uses Machine Learning to classify iris flowers into:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

## Model Used
K-Nearest Neighbors (KNN)

## Accuracy
~95%+

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python main.py
